## 3. 响应式断点

### 3.1 设备适配

```
320px - 375px  (小屏手机)
├─ 字体: 48px
├─ 间距: 紧凑
└─ 卡片: 单列

376px - 414px  (标准手机)
├─ 字体: 56px
├─ 间距: 标准
└─ 卡片: 单列

415px - 768px  (大屏手机/小平板)
├─ 字体: 64px
├─ 间距: 宽松
└─ 卡片: 可选2列

768px+  (平板/桌面)
├─ 字体: 72px
├─ 间距: 宽松
└─ 卡片: 多列
```

### 3.2 字体大小自适应

```
vw单位计算:
主数字: 15vw (最大72px, 最小48px)
标签文字: 4vw (最大18px, 最小14px)
信息文字: 5vw (最大20px, 最小16px)
```

## 6. 无障碍设计

### 6.1 屏幕阅读器标记

```
<主要区域>
  role="main"
  aria-label="时间显示"
  aria-live="polite"
  aria-atomic="true"
  
<进度条>
  role="progressbar"
  aria-valuenow="50"
  aria-valuemin="0"
  aria-valuemax="100"
  
<按钮>
  role="button"
  aria-pressed="false/true"
  aria-label="切换主题"
```

### 6.2 键盘导航（外接键盘）

```
Tab键顺序:
1. Logo
2. 时区选择
3. 语言选择
4. 主题切换
5. 主显示区
6. 信息卡片

快捷键:
Space - 暂停/继续
T - 切换主题
L - 切换语言
Z - 选择时区
R - 调整刷新率
```

```

1. HTML主体 (3KB)
```
